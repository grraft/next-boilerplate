# next-boilerplate
A starter project for building Next.js apps featuring;
- [ ] React              : view
- [ ] Redux              : state
- [ ] Next.js            : ssr, routing
- [ ] Jest               : tests
- [ ] Istanbul           : coverage
- [ ] styled-jsx         : style
- [ ] SASS               : pre-processor
- [ ] babel es6          : js
- [ ] build script       : production
- [ ] deploy instruction : AWS, CI
